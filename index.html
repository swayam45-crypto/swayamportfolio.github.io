Projects

I built a breadboard circuit using a multiplexer. The purpose of the circuit was to select one input signal from multiple inputs and send it to a single output line. I connected the input lines, selection pins, and output through jumper wires on the breadboard. The select lines were used to choose which input to pass through. This helped me understand how multiplexers work in data selection and signal routing in digital circuits.
